# web-next — Next.js 14

Frontend principal del stack. Consume datos de la API Python.

## Scripts

```bash
npm run dev     # Dev server en localhost:3000
npm run build   # Build de producción
npm run start   # Servir build en localhost:3000
```

## Estructura

- `pages/index.js` — Página principal que hace fetch a `http://localhost:3004/`
- `next.config.js` — Configuración mínima
